# 🎬 Alura Plus

O **Alura Plus** é um projeto de página web desenvolvido com HTML e CSS que simula a landing page de uma plataforma de streaming educacional.

O objetivo do projeto é praticar a criação de interfaces modernas para web, trabalhando com organização de layout, estruturação de conteúdo e estilização visual.

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* **HTML5**
* **CSS3**

# 🎯 Objetivos do Projeto

* Praticar a estruturação de páginas com **HTML semântico**
* Aplicar **estilização e organização de layout com CSS**
* Desenvolver uma interface clara e visualmente organizada
* Simular uma página de divulgação de produto digital
