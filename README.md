# 📚 Alura — Projetos de Desenvolvimento Web

Repositório destinado à organização e apresentação dos projetos desenvolvidos durante cursos de desenvolvimento web realizados na plataforma **Alura**.

Este repositório reúne exercícios e projetos práticos desenvolvidos ao longo das aulas, com foco no aprendizado e aplicação de tecnologias web.
