# 📱 Apeperia — Layout Responsivo

Projeto de uma página web com layout responsivo desenvolvido durante o curso **"Layouts Responsivos: Trabalhando com layouts mobile"** da plataforma Alura.

O projeto tem como objetivo praticar a construção de interfaces modernas utilizando **design responsivo**, garantindo uma boa experiência de navegação em diferentes tamanhos de tela, especialmente em dispositivos móveis.

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* HTML5
* CSS3
* Emmet (para otimização da escrita do código)

# 🎯 Objetivos do Projeto

* Praticar construção de layouts responsivos
* Aplicar conceitos de **Mobile First**
* Organizar estrutura HTML de forma semântica
* Utilizar **CSS para adaptação de layout em diferentes resoluções**
