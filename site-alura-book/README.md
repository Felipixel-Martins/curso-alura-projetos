# 📚 AluraBooks

O **AluraBooks** é um projeto de interface web que simula uma livraria online especializada em livros sobre tecnologia.

O objetivo do projeto é praticar conceitos de **desenvolvimento front-end**, incluindo estruturação de páginas com HTML, estilização com CSS e criação de carrosséis interativos utilizando SwiperJS.

---

# 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

* HTML5
* CSS3
* SwiperJS

# 🎯 Objetivos do Projeto

* Praticar estruturação de páginas com **HTML semântico**
* Aplicar **estilização e layout com CSS**
* Implementar **componentes interativos utilizando SwiperJS**
* Desenvolver uma interface moderna e responsiva
